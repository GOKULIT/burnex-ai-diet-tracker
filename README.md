Burnex – AI Diet Planner & Food Tracker

Burnex is an AI-powered diet planner and food tracking web application.

It allows users to upload food images and track their daily diet efficiently.

Features
- Food image upload
- Diet tracking system
- Daily monitoring
- User-friendly interface

Tech Stack
- Python
- HTML, CSS, JavaScript
- API Integration
